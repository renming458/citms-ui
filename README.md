
## Install
```shell
npm install ctims-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import ctims from 'ctims-ui'

Vue.use(ctims)

// or
import {
  Select,
  Button
  // ...
} from 'ctims-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```

## Browser Support
Modern browsers and Internet Explorer 10+.
